# spotify-music-analytics

The Spotify Music Analytics Dataset (2015-2025) provides a comprehensive view of global music trends from 2015 to 2025, containing 85,000 tracks with detailed metadata and audio features. Each record includes track names, artists, albums, genres, release dates, tempo, loudness, energy, danceability, instrumentalness, valence, popularity scores, streaming counts, country, and label information. It covers a wide range of audio features, popularity metrics, streaming statistics, and metadata. The dataset itself is synthetic, meaning it is data generated by an algorithm rather than observed from real-world measurements, for example by generative AI technologies. The dataset was carefully generated to ensure: Balanced distribution across genres, years, and popularity levels, Consistency in numeric audio features (danceability, energy, loudness, etc.), Realistic patterns in categorical features (genre, artist, label, country), Safe public use — fully synthetic, no copyrighted content. Essentially, the data explores how genres, popularity, and audio features evolve over a decade.
The dashboard was built in Streamlit to allow stakeholders to:
- Filter by country, year,  and musical key
- Compare genres
- Analyze streaming trends
- Examine relationships between popularity and audio features
- Identify high-performing labels

The primary stakeholders for this analysis are Spotify's product-related and content/marketing strategy teams, who would benefit from understanding how listener preferences and music track characteristics relate to popularity and streaming performance over time. Specifically, the stakeholders would want us to identify patterns in music genre trends, audio features, and regional differences that can inform playlist curation, content promotion/marketing, and data-driven song or artist recommendations.
The questions our stakeholders would seek to answer:
- How have genres and audio features associated with popular tracks changed from 2015 to 2025?
- What relationships exist between audio features and popularity or streaming counts?
- Are there any differences to note in music trends across countries or labels?
- How do popularity and streaming metrics for newly released content vary across genres, to guide future steps for promotional reach and visibility?
- Which music label produces the most popular music?
These stakeholders would use the dashboard to:
- Inform playlist curation strategies
- Guide promotional targeting by region
- Identify emerging genres
Overall, the primary stakeholders would leverage the dashboard to gain specific insights about up-and-coming Music Labels, Genres, Musical Keys, all in relation to popularity and stream count for countries over time.
